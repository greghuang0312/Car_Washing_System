# 管理员账号管理手册

> 适用场景：更换管理员微信账号、新增管理员、移除管理员权限

---

## 核心原理

系统使用微信 `openid` 作为管理员身份标识，所有有效管理员的 openid 存储在云数据库 `admins` 集合中。  
登录时小程序自动读取当前微信账号的 openid，与 `admins` 集合比对，匹配则放行。

---

## 操作一：更换管理员（交接给业主）

### Step 1：获取业主的 openid

**方法**：让业主用他的微信打开小程序（扫体验版或正式版二维码），然后去：

> 云开发控制台 → 云函数 → `adminLogin` → **日志**

找到业主的调用记录，日志中会显示 `"openid": "oXXXX..."` 字段，复制该值。

> **提示**：如果 `adminLogin` 云函数日志中 openid 字段未显示，可临时在云函数日志里查看所有请求记录。

---

### Step 2：更新 admins 集合

1. 打开云开发控制台 → **数据库** → `admins` 集合
2. 找到当前管理员记录，点击编辑
3. 将 `openid` 字段的值替换为业主的 openid

或者：删除旧记录，重新添加：
```json
{
  "openid": "业主的openid",
  "name": "业主姓名"
}
```

---

### Step 3：验证

业主用他的微信打开小程序 → 点击「管理入口」→ 确认能正常进入管理后台（无需密码）。

---

## 操作二：新增管理员（多人管理）

直接在 `admins` 集合中**添加一条新记录**即可，系统支持多个 openid 同时具备管理员权限。

```json
{
  "openid": "新管理员的openid",
  "name": "张三（助手）"
}
```

获取新管理员 openid 的方法同 Step 1。

---

## 操作三：移除管理员权限

在 `admins` 集合中找到对应记录，**直接删除**即可。该账号立即失去管理员资格。

---

## 管理员权限说明

具备管理员权限可进行以下操作：

| 功能 | 说明 |
|---|---|
| 查看所有订单 | `getOrders` 云函数，分页查询全部用户订单 |
| 查看统计数据 | `getStats` 云函数，按日/周/月/全部统计 |
| 设置洗车价格 | `updatePrice` 云函数，更新每分钟计费金额 |
| 生成测试数据 | `generateMockData` 云函数，仅限测试环境使用 |

---

## 常见问题

**Q：业主登录后提示「无管理权限」怎么办？**  
A：openid 未正确写入 `admins` 集合。检查 openid 是否有多余空格，或大小写是否一致。

**Q：管理员换了手机会影响登录吗？**  
A：不影响。openid 与微信账号绑定，与手机设备无关，换手机无需重新配置。

**Q：如何确认当前登录账号的 openid？**  
A：在开发者工具 Console 中运行：
```js
wx.cloud.callFunction({ name: 'adminLogin', data: {} }).then(r => console.log(r.result.openid))
```
